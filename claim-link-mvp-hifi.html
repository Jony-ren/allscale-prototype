  • Fees = network reserve（receiver gasless）+ service fee (§5.2.1)
  • 取消语义全面改 "Claim back to my wallet"（TipLink 学4）；状态机内部仍是 cancelled (§5.4)
  • States: pending → link_sent → claiming → claimed/expired/cancelled (§5.4)；过期时长已定 2 周（0611）
  • 功能 1 不设单笔上限：同现有 send，每笔本人验证码确认；限额只存在于商家批量授权域（API 免逐笔确认才需要，见 Merchant tab）
  • Claiming locks Claim back — try the state buttons (§5.4)；link re-send fission deferred to Phase 2 (0610)

MVP · Unified claim
  • Public page, no login (§5.2.1)
  • USD-first display; token amount secondary (0610, TipLink 学3)
  • Expandable [Claim] CTA — options appear on tap, recommended first (学1)
  • Shown amount = received amount, no hidden deductions (§8.2, 戒1)
  • UI pushes Open AllScale Wallet to onboard (Q2); path A success page upsells account (学2)
  • Path A: direct claim, no OTP (confirmed) — leak risk borne by link secrecy + limits (§5.2.2)
  • Path B 两步交互：确认邮箱 → [Send code] → 6 位验证码自动跳格（0611）
  • 邮箱解锁：prefill 收件邮箱但可修改——A 邮箱收链接可领进 B 邮箱的账户；OTP = 账户鉴权，非收款人校验
  • 已登录直通：有会话时两项为 AllScale / 非 AllScale 二分——「Claim into my AllScale Wallet · 1-click」+「Withdraw to External Wallet（0x）」（有账户即有钱包，不出现"开钱包"和"no signup"话术）；A 收 B 领降级为小链接
  • 成功页按路径区分：路径 A（外部地址）才出现开户引导；路径 B / 已登录领取完成时本就已登录，显示「Go to my AllScale Wallet」
  • 0x check + irreversible confirm (§8.2)
  • Down: funds safe, temporarily unclaimable (§6.8)
